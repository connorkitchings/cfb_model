# Points-For Modeling Scripts (Archived)

Archived on: 2025-11-22

## Reason for Archival

Points-for modeling may be refactored separately in the future. These scripts are preserved for reference or potential restoration.

## Scripts

- `train_points_for_models.py` - Train points-for prediction models
- `build_points_for_slice.py` - Prepare points-for data slices
- `run_points_for_feature_selection.py` - Feature selection for points-for
- `analyze_points_for_calibration.py` - Calibration analysis
- `combine_points_for_and_legacy.py` - Legacy data migration

## Restoration

To restore:

```bash
cp scripts/archive/points_for/*.py scripts/
```
