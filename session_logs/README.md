# Session Logs

This directory contains the session logs for the Vibe Coding project, organized by date.

## Structure
`YYYY-MM-DD/NN-title.md`

- **YYYY-MM-DD**: Date of the session.
- **NN**: Sequence number for that day (e.g., 01, 02).
- **title**: Brief slug derived from the sprint objective.

## Usage
These logs are automatically generated by the MADE orchestrator when starting a new sprint via `vibe sprint start`.
